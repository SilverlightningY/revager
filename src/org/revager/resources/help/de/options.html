Programm-Einstellungen

Öffnen Sie über die Menüleiste unter Extras das Fenster "Programm-Einstellungen".

Die Programmeinstellungen gliedern sich in die folgenden Register:

1) Allgemein:

Abb.Programmeinstellungen- Allgemein

Hinweis: In diesem Fenster können alle zentralen Programmeinstellungen vorgenommen werden. Diese werden anschließend in der Datenbank gespeichert und sind unabhängig vom Review.

Folgende das komplette Programm betreffende Optionen sind in diesem Bereich konfigurierbar:

  • Sie können hier entscheiden, ob beim Start des Programms geprüft werden soll, ob eine neuere Version der Software verfügbar ist.
  • Möchten Sie eine automatische Speicherung eines geöffneten Reviews in regelmäßigen Abständen, können Sie hier entscheiden in welchem zeitlichen Intervall die Review-Datei automatisch gespeichert werden soll und ob die automatische Speicherung für das ganze Programm oder nur während der Protokollierung aktiviert werden soll.
    •

    2) PDF-Export:

    Abb.Programmeinstellungen- PDF-Export

    Hinweis: Sie können hier einige Einstellungen bezüglich des PDF-Exports vornehmen. Diese werden anschließend in der Datenbank gespeichert und sind unabhängig vom Review.

    Für den PDF-Export sind folgende Optionen zur Konfiguration verfügbar:

  • Sie können hier bei Bedarf ein Logo für alle Review- und Sitzungsprotokolle im PDF-Format hochladen und einen Fußtext bestimmen, der eingeblendet werden soll.
  • Für die Review-Einladungen im PDF-Format können Sie hier ebenfalls ein Logo hochladen und einen Einladungstext, sowie einen Fußtext bestimmen, der eingeblendet werden soll.
