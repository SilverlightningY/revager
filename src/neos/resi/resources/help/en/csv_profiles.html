Profile für den CSV-Export

Öffnen Sie über die Menüleiste unter Extras das Fenster "Profile für den CSV-Export".

Abb.Profile für den CSV-Export

Hinweis: In diesem Fenster können Sie die unterschiedlichen Profile für den Export von protokollierten Befunden im CSV-Format konfigurieren.

Hier können Sie nun die unterschiedlichen Profile für den Export von protokollierten Befunden im CSV-Format konfigurieren.

Folgende Optionen sind konfigurierbar:

  • Sie können den Namen des CSV-Profils festlegen.
  • Die Reihenfolge der Spalten in der CSV-Datei kann hier bestimmt werden.
  • Die Bezeichnungen der Spalten in der CSV-Datei können ebenfalls festgelegt werden.
  • Zudem können Sie für das Profil gültige Gewichtungen der Befunde angeben.
