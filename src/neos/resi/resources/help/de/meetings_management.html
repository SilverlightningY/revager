Um eine Sitzung anzulegen, zu bearbeiten, zu entfernen, zu kommentieren oder abzusagen, muss ein Review geöffnet oder ein neues Review angelegt worden sein. Das heißt Sie müssen sich im Hauptfenster des zuvor ausgewählten Modus befinden.

Neue Sitzung anlegen

Im Hauptfenster befindet sich eine Auflistung der Sitzungen.

Abb.Auflistung der Sitzungen

Um eine neue Sitzung anzulegen klicken Sie hier:
Sie können die Funktion "Neue Sitzung" auch aus der Werkzeug- oder Menüleiste wählen.

Es erscheint das Fenster "Sitzung hinzufügen".

Abb.Sitzung hinzufügen/bearbeiten

Hinweis: Wird eine neue Sitzung angelegt, so sind alle Felder leer und müssen von Ihnen ausgefüllt werden. Andernfalls sind die Felder mit den Werten der gewählten Sitzung bereits ausgefüllt und können bearbeitet werden.
Im Textfeld "Datum" wird das geplante Sitzungsdatum eingetragen. Als Alternative zur manuellen Eingabe des Datums kann neben dem Textfeld ein Kalender eingeblendet werden mit Hilfe dessen ein Datum ausgewählt werden kann.
Im Textfeld "Zeitraum" werden die geplanten Uhrzeiten eingetragen, wann die Sitzung beginnen und wann sie enden soll.
Im Textfeld "Ort" wird erfasst, wo die Sitzung stattfinden soll.
Beim Bearbeiten einer Review-Sitzung kann ein Haken bei "Abgesagt" gesetzt werden. Dies bedeutet, dass die Sitzung abgesagt wurde. Die Begründung der Absage sollte im nebenstehenenden Textfeld eingetragen werden.

Bitte tragen Sie im Folgenden die geforderten Daten zur neuen Sitzung zusammen und bestätigen Sie die Eingaben anschließend.

Sitzung bearbeiten

Im Hauptfenster befindet sich eine Auflistung der Sitzungen. Wählen Sie bitte eine Sitzung aus der Liste.
Um die Sitzung zu bearbeiten klicken Sie hier:
Das Fenster "Sitzung bearbeiten" erscheint (Siehe Abb.Sitzung hinzufügen/bearbeiten).

Bereits bestehende Daten können nun über die entsprechenenden Eingabefelder bearbeitet werden. Bitte bestätigen Sie anschließend Ihre Eingaben.

Sitzung kommentieren

Im Hauptfenster befindet sich eine Auflistung der Sitzungen. Wählen Sie bitte eine Sitzung aus der Liste.
Um die Sitzung zu kommentieren klicken Sie hier:

Es öffnet sich nun ein Text-Popup Fenster, mit dessen Hilfe die gewählte Sitzung kommentiert bzw. ein bereits vorhandener Kommentar bearbeitet werden kann.
Bitte bestätigen Sie anschließend Ihre Eingabe.

Sitzung absagen

Im Hauptfenster befindet sich eine Auflistung der Sitzungen. Wählen Sie bitte eine Sitzung aus der Liste.
Klicken Sie nun hier:
Das Fenster "Sitzung bearbeiten" erscheint (Siehe Abb.Sitzung hinzufügen/bearbeiten).

Um eine Sitzung abzusagen wählen Sie nun die Absage-Option und geben den Grund für die Absage in das dafür vorgesehene Freitextfeld ein.

Bitte bestätigen Sie die Eingabe.
