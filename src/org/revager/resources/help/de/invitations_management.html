Einladungspakete erzeugen


Um Einladungspakete zu erstellen muss mindestens eine Sitzung angelegt worden sein, welche aus mindestens einem Teilnehmer besteht.

Wählen Sie nun über die Werkzeug- oder Menüleiste die Funktion "Einladungspakte erstellen".

Das Fenster "Einladungspakete erzeugen" erscheint.

Abb.Einladungspakete erzeugen

Anschließend wählen Sie bitte eine Sitzung.
Aus der Auflistung der Sitzungsteilnehmer wählen Sie bitte diejenigen aus, für welche Sie ein Einladungspaket erstellen möchten.

Optional können Sie noch Dateien, welche den Prüfling repräsentieren, an das Einladungspaket anhängen.

Im Folgenden müssen Sie noch angeben, in welcher Form die Einladungspakete erstellt werden sollen, dass heißt als einzelne PDF-Dateien, als ZIP-Archive oder als Verzeichnisse.

Wählen Sie nun einen Speicherort für die Einladungspakete im Dateisystem aus, in dem die Einladungspakete erstellt werden sollen.
